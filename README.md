# Anticor2rss
Génère un flux rss pour les articles d'[Anticor](https://www.anticor.org/articles/)

### Usage
```bash
$ ./anticor2rss.sh
```

Flux rss disponible [ici](https://www.remigarcia.xyz/rss/anticor.rss) et mis à jour tous les jours à minuit.
